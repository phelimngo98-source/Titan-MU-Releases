## 📖 Giới thiệu

**PhelimN Titan Manager** là ứng dụng quản trị server MU Online dành cho hệ thống **Titan Server**, với hiệu suất cao và giao diện hiện đại.

Ứng dụng giúp Admin quản lý toàn bộ các khía cạnh của server MU Online chỉ với vài thao tác click chuột — không cần phải mở SQL Server Management Studio hay chỉnh sửa file thủ công.

---

## ✨ Tính năng chính

### 👤 Quản lý Tài khoản
- Tạo mới, sửa, xóa tài khoản
- Khóa / mở khóa tài khoản
- Đổi tên tài khoản
- Quản lý VIP (kích hoạt, tìm kiếm theo tài khoản/nhân vật)
- Quản lý tiền tệ tài khoản (WCoin, Zen...)

### 🎮 Quản lý Nhân vật
- Xem danh sách nhân vật, tìm kiếm theo tài khoản
- Tạo nhân vật mới, đổi tên, xóa, khóa/mở
- Chỉnh sửa thông số nhân vật (stats, level, class...)
- Reset Grand & Master Resets

### 🎒 Quản lý Kho đồ (Inventory & Warehouse)
- **Kho đồ nhân vật:** Xem, sửa, thêm/xóa item trực quan với giao diện kéo thả
- **Kho ngân hàng:** Quản lý warehouse chính và các kho mở rộng
- **Chỉnh sửa item chi tiết:**
  - Level, Option, Skill, Luck
  - Tính năng hoàn hảo (Excellent Options)
  - Ngọc hài hòa (Jewel of Harmony)
  - Socket Options (5 slot)
  - Đồ thần (Ancient Set Option)
  - Hạn dùng item (Period Item) — đặt và gia hạn thời gian
- **Tìm kiếm item** theo tên và ID
- **Hoán đổi item** — đổi item sang item khác
- Hình ảnh item hiển thị trực tiếp từ file Data

### 💎 Quản lý Ngọc (Jewels)
- Thống kê số lượng ngọc theo tài khoản
- Bảng xếp hạng ngọc
- Chỉnh sửa số lượng ngọc cho tài khoản

### 💰 Quản lý Tài chính
- Dashboard thống kê doanh thu tổng quan
- Biểu đồ doanh thu theo thời gian
- Quản lý giao dịch nạp tiền
- Phân chia lợi nhuận giữa các cổ đông
- Bảng xếp hạng WCoin / Zen
- Báo cáo tài chính chi tiết
- Mục tiêu tài chính (Financial Target)

### 🎁 Hệ thống Gift Code
- Tạo và quản lý Gift Code hệ thống
- Thiết lập item phần thưởng cho từng mã
- Xóa Gift Code đã hết hạn

### 🏆 Hệ thống Nạp thưởng (Donation Milestones)
- Tạo các mốc nạp thưởng
- Thiết lập item phần thưởng cho từng mốc
- Quản lý lịch sử nạp của tài khoản
- Nhận thưởng và reset nạp

### ⚙️ Cài đặt & Tiện ích
- Kết nối nhiều server cùng lúc (Multi-Server)
- Tự động kết nối DB khi khởi động
- Cập nhật tự động (Auto-Updater)
- Giao diện tối (Dark Mode)
- Đa ngôn ngữ (Tiếng Việt / English)
- Import/thay thế file `Item.txt`
- Thay đổi resolution

## 📥 Cài đặt

### Người dùng mới
1. Tải file **Setup** từ [Releases](https://github.com/phelimngo98-source/Titan-MU-Releases/releases/latest)
2. Chạy file cài đặt và làm theo hướng dẫn
3. Khởi động ứng dụng → Cấu hình kết nối server

### Người dùng cũ (Cập nhật)
- Ứng dụng **tự động kiểm tra** bản cập nhật khi khởi chạy
- Nếu có phiên bản mới → hiện thông báo → tự tải về và cài đặt
